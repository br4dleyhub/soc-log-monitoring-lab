## Alert Rules

### Alert 1 – Possible Brute Force Attack
Trigger when:
- More than 2 failed login attempts
- Followed by a successful login
- Within a short time window

### Alert 2 – Suspicious File Modification
Trigger when:
- A file is modified
- Shortly after a successful login
- By the same user

### Alert 3 – Repeated Network Access
Trigger when:
- Multiple connections
- From the same IP
- To a sensitive port

## Correlation Logic

Alerts are generated by linking authentication,
file access, and network events based on:
- User identity
- IP address
- Time proximity
